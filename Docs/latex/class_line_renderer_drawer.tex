This class gets a line renderer in this object, and then set its end points based on its target. 

Explanation\+:

Usage\+: Integration\+:
\begin{DoxyItemize}
\item Attach this class script to an Obj with Line\+Renderer attached to it.
\item Line Renderer will automatically set its end\+Point as the target that is passed in this script
\item You can swap the target end\+Point by calling the set\+Target function
\item You can reset the target to the initial default target by calling the Reset\+Target function
