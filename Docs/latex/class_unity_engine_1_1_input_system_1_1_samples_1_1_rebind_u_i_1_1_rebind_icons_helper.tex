\doxysubsection{Detailed Description}
This is an example for how to override the default display behavior of bindings. 

The component hooks into \mbox{\hyperlink{class_unity_engine_1_1_input_system_1_1_samples_1_1_rebind_u_i_1_1_rebind_action_u_i_ae6f650d471a43cb2f5087df933289a39}{Rebind\+Action\+UI.\+update\+Binding\+UIEvent}} which is triggered when \mbox{\hyperlink{namespace_unity_engine_1_1_input_system_1_1_u_i}{UI}} display of a binding should be refreshed. It then checks whether we have an icon for the current binding and if so, replaces the default text display with an icon.
